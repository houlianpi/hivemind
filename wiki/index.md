# Hivemind 知识库

> 团队知识积累，基于 LLM Wiki 模式

欢迎来到 Hivemind！这是一个团队知识库，收集和整理 AI 编程工具、技巧和最佳实践。

---

## 🛠️ 工具介绍

### AI 编程助手

- **[OPC - One Person Company](/opc)** — Claude Code 技能，内置 11 个专业角色（PM、设计师、安全工程师等），帮你审查、分析、构建代码

### Copilot API 代理

用 GitHub Copilot 订阅驱动 Claude Code、Codex CLI 等工具：

- **[Copilot Gateway](/copilot-gateway)** — 轻量级 API 代理，部署在 Deno Deploy / Cloudflare Workers
- **[Copilot API Proxy](/copilot-api)** — 功能丰富的代理，带使用量仪表盘、速率限制、手动审批
