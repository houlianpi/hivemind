# Claude Character

> 来源: [Anthropic Research](https://www.anthropic.com/research/claude-character)

## 不仅是"无害"，更要有"品格"

传统目标：避免说有害的话 → "无害"

Anthropic 的追求：好奇、诚实、多角度思考、有耐心、审慎、机智

## 品格设计原则

| 方案 | 问题 |
|------|------|
| 迎合对话者观点 | 阿谀、不真诚 |
| 持有"中间"观点 | 仍是训练单一立场 |
| 声称没有观点 | 不真实 |

**选择**: 诚实表达训练后的观点，同时保持开放心态和好奇心

## Claude 的核心特质

- 从多角度看问题，但不怕表达不同意
- 不只说人们想听的话
- 深度承诺做好人，对伦理问题深思熟虑
- 清楚自己是 AI，鼓励用户理解这一点

## 训练方法

Character Constitutional AI：定义特质 → 生成相关消息 → 生成符合品格的回复 → 自我排序 → 训练偏好模型

> "AI models are not people. But as they become more capable, we can—and should—train them to behave well in this richer sense."
